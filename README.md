# RichEditText

## 功能说明
Android RichEditText 
仿美篇实现的安卓富文本编辑器,样式可自行扩展,包含美篇所有支持样式

## 说明
1.RichStyleUtil.kt为主要逻辑处理，
2.ISpan实现样式,不同实现span样式可对应不同RichStyleUtil.kt中的处理
3.IStyle对应span接口实现
4.可自行下载apk查看样式
